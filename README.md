## 🌟 Overview
AuraHealth AI is a professional-grade diagnostic tool designed to bridge the gap in mental health accessibility. Using the **DistilBERT Transformer** architecture, it provides high-accuracy emotional analysis and actionable clinical recommendations. 

The project aligns with **UN Sustainable Development Goal #3 (Good Health and Well-being)** and is designed with **IBM Z Pervasive Encryption** principles in mind to protect sensitive psychological data.

## ✨ Key Features
- **Neural Emotion Detection:** Recognizes 6 primary emotional states using Deep Learning.
- **Confidence Scoring:** Transparent AI that signals when data is insufficient for a reliable diagnosis.
- **Actionable Recommendations:** Immediate coping mechanisms based on detected stress levels.
- **Clinical Reports:** Exportable session summaries for personal tracking or professional use.
- **Privacy-First:** Local neural inference ensures raw text never leaves the secure environment.

## 🛠 Tech Stack
- **Language:** Python 3.12
- **Frameworks:** Streamlit, PyTorch
- **AI/ML:** Transformers (Hugging Face), DistilBERT
- **Data Visualization:** Altair, Pandas
- **Security Logic:** IBM Z Pervasive Encryption Standards
